# Desktop Agents 使用指南

> 持续更新文档。最后更新：2026-05-23

Desktop Agents 是一个轻量桌面多 Agent 聊天应用。当前版本已精简掉工具执行、权限弹窗和截图能力，重点保留桌面聊天、人格切换、聊天记录和 API Key 配置。

## 1. 启动项目

在项目目录运行：

```powershell
py F:\xmlg\agent\desktop-agents\main.py
```

启动后会出现多个桌面 Agent。第一次使用时，如果没有配置 API Key，会先弹出配置窗口。

## 2. 配置大模型 Key

### 首次配置

第一次启动时会自动检查是否已有 Key：

- 如果存在 `LLM_API_KEY` 或 `DEEPSEEK_API_KEY`，直接启动。
- 如果没有，会弹出“首次使用配置”。

需要填写：

| 字段 | 示例 |
| --- | --- |
| Provider | `deepseek` / `openai` / `claude-compatible` |
| Base URL | `https://api.deepseek.com/v1` |
| Model | `deepseek-chat` |
| API Key | 你的模型 API Key |

### 随时修改

右键任意 Agent，点击：

```text
API Key 设置
```

保存后会热更新当前 Agent 的模型客户端，后续聊天直接使用新配置。

## 3. 基本交互方式

### 和 Agent 说话

左键点击 Agent，输入你想说的话。

例如：

```text
大家帮我分析一下下一步应该优先做什么
```

Agent 会在桌面气泡里回复。

### 打开群聊记录

右键 Agent，点击：

```text
聊天记录
```

可以查看最近的多 Agent 对话，也可以清空记录。

### 多 Agent 自动群聊

项目启动后，多个 Agent 会自动进行轻量群聊。你可以随时插话，系统会选择 1-2 个 Agent 回复。

## 4. 人格切换

右键 Agent，在“切换人格”菜单里选择其他人格。

切换后：

- 当前 Agent 会应用新人格名称、风格、头像等配置。
- 当前 Agent 的内部聊天历史会清空。
- 群聊窗口会继续保留整体最近记录。

## 5. 导入聊天记录人格

右键 Agent，点击：

```text
导入聊天记录人格
```

支持：

- 单文件导入：`导入聊天记录文件`
- 文件夹导入：`批量导入`

支持格式：

```text
txt / csv / json / sqlite / db / sqlite3
```

如果不知道怎么导出聊天记录，界面里会提示参考：

```text
https://github.com/r266-tech/wechat-local-mcp
```

项目本身不自动下载第三方工具。

## 6. 系统托盘

系统托盘已保留为最小控制入口，只包含：

- 显示
- 隐藏
- 退出

API Key 设置、聊天记录和导入入口保留在 Agent 右键菜单里，不再放到托盘里。

## 7. 当前已移除能力

当前精简版已移除以下过度设计能力：

- 工具执行框架
- 权限分级系统
- 工具权限确认弹窗
- 读文件工具
- 执行命令工具
- 截图工具
- 模型 tool_calls 调用循环

因此，桌面 Agent 现在只负责聊天和人格交互，不会读取文件、执行命令或截图。

## 9. 安全建议

- 不要把 API Key、私钥、密码文件内容直接发给 Agent。
- 当前版本不会主动读取本地文件，也不会执行本地命令。
- API Key 优先保存到系统凭据管理器；如果不可用，会 fallback 到当前 Windows 用户配置。

## 10. 后续可继续精简项

可以继续考虑：

- 是否保留聊天记录导入人格
- 是否保留聊天记录窗口
- 是否继续简化 API Key / 多模型配置
- 是否减少默认 Agent 数量
- 是否整理人格配置文件
